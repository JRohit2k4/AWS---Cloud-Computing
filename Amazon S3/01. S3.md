**Amazon S3**  

**Amazon S3 (Simple Storage Service)** is one f the core building block of AWS and provides **infinitely scalable object storage**. Many websites and AWS services rely on S3 as a backend for storing and retrieving data.

---

**Key Charasteristics**

- Highly scalable and durable storage
- Used widely across AWS services
- Appears global, but buckets are region-specific
- Stores data as objects inside buckets

---

**Common Use Cases**  
Amazon S3 is used for:  
- Backup and Storage (files, disks, data)
- Disaster recovery (cross-region data backup)
- Archival storage (low-cost long-term storage using S3 Glacier)
- Hybrid cloud storage (on-prem + cloud)
- Hosting applications and media (images, videos)
- Data lakes and big data analytics
- Software delivery and updates
- Static website hosting

**Real-world examples:**
- NASDAQ stores 7 years of data in S3 Glacier
- Sysco runs analytics on S3 data to gain business insights

---

**Buckets**
- Buckets are top-level containers in S3
- Bucket names must be globally unique across all AWS account
- created in specific AWS region
- S3 is global in service scope, but bucket location is regional (common beginner mistake)
