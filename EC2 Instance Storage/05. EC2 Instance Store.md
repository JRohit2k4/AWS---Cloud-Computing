**EC2 Instance Store**

**What is EC2 Instance Store**
- EC2 Instace store is a local storage physical attached to the host machine
- It is hardware disk storage, not network-based
- Also called **Ephemeral Storage**

Think of it as:  
A local SSD/HDD inside the physical server running your EC2 instance

---

**Why Use Instance Store?**
- Extremely high performance
  - Very low latency
  - Very high IOPS

- Better performance than EBS because:
  - No network involved
  - Direct hardware access

**Best Use Cases**
- Instance Store is ideal for:
- Cache
- Buffers
- Scratch space
- Temporary files
- High-performance workloads

**Not suitable for:**
- Databases
- Long-term storage
- Critical data
