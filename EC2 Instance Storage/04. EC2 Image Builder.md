**EC2 Image Builder**

**What is EC2 Image Builder?**
- EC2 Image Builder is a service that automates creation, maintenance, testing, and distribution of AMIs
- It removes the need of manually build and update AMIs

Exam-safe definition:
- EC2 Image Builder automates AMI creation, maintenance, testing, distribution and keeps them secure and up to date.

---

**What Problem does It Solve?**
Without Image Builder:
- Manually launch EC2
- Install software
- Patch OS
- Create AMI
- Repeat every update

**With Image Builder**
- Fully automated
- Repeatable
- Consistent
- Less human error

---

**How EC2 Image Builder Works**
1. Image Builder runs
2. AWS creates a temporary Builder EC2 instance
3. Components are installed:
  - OS Updates
  - Softwares
  - Security patches
  - Applications
4. An AMI is created
5. (Optional) A test EC2 instance is launched
  - Runs validation tests
6. AMMI is distributed
  - Same region
  - Multiple region
  
All of this is automated

---

**Scheduling**
EC2 Image Builder can run:
- On a schedule (weekly, monthly)
- When packages change
- Manually

Exam keyword:
"Automatically keep AMIs up to date" -> EC2 Image Builder

---

**Cost Model**
- EC2 Image Builder itself is **FREE**
- You pay only for:
  - EC2 instances it launches
  - AMI storage
  - Data transfer (if applicable)
 
If you see:
"No additional service charge" -> EC2 Image Builder

---

**Region & Distribution**
- AMIs are region-based
- EC2 Image Builder can:
  - Create AMIs
  - Distribute them across multiple regions

This supports global deployments.
